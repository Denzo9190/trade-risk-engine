# Contributing

This project is currently maintained as a personal engineering lab.

However, suggestions, architecture discussions and improvements are welcome.

Please open an issue for:

- architecture discussions
- performance improvements
- financial math correctness
- risk engine ideas
  Architecture Decision Records (ADR)

ADR files document stable architectural decisions
that are implemented in the system.

ADR should be added to the repository only after
the decision has been implemented and verified.

Intermediate discussions remain in the project
decision log.
